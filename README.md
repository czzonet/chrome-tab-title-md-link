# Chrome Tab Title Md Link

> Chrome 小插件：点击提取网页标题和链接，拼接用于 md 文档的链接格式。

## 功能

- [x] 单独列出当前标签页
- [x] 列出所有标签页
- [x] 复制到剪贴板
